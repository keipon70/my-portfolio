🚀実績紹介

Googleフォーム問い合わせのLINE即時通知システム

課題：フォームへの問い合わせに気づくのが遅れ、顧客対応が後手に回ってしまう。
解決策：Googleフォームに新しい回答が送信された際、その内容をリアルタイムで指定のLINEにプッシュ通知する自動化システムを開発。担当者はスマホで手軽に、即座に内容を確認できるようになりました。

使用技術： Google Apps Script, LINE Messaging API

OpenAI(ChatGPT)連携 LINEチャットボット

課題：よくある質問に人手が割かれ、コア業務に集中できない。
解決策：LINEに送られたメッセージに対し、ChatGPTが文脈を理解して自動返信するAIチャットボットを開発。24時間365日、一次対応を自動化し、カスタマーサポートの工数を大幅に削減しました。

使用技術： OpenAI API, LINE Messaging API, Python

ノーコードAIチャットボット構築 (Dify)

課題：プログラミング不要で、特定の知識を学習させたAIチャットボットを迅速に導入したい。
解決策：ノーコードプラットフォーム「Dify」を利用し、FAQや商品情報などを学習させたAIチャットボットを短期間で構築。社内ヘルプデスクやキャンペーン対応を効率化しました。

使用技術： Dify, 大規模言語モデル(LLM)

LP(ランディングページ)制作・公開

課題：新サービスやキャンペーンの顔となるWebページを、サーバー契約なしで迅速に立ち上げたい。
解決策：HTML/CSSを用いてLPをコーディングし、GitHub Pagesを利用して無料で公開。デザインから公開までを一気通貫で対応しました。

使用技術： HTML, CSS, GitHub Pages
